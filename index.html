My Projects

Sonata

Sonata is an elegant android application that allows users to listen and create playlists of songs from YouTube. Sonata is a multitasking app, allowing the user to listen while using other apps. Winning hack in a Velocity sponsored hackathon.

Find

Find is an Android application that allows a user to locate his/her friends by entering their unique Find IDs. Find is a great way for a group of friends to locate each other when in an unfamiliar setting.

Github

FX News

FX News is an Android application that displays Forex News headlines in a minimalistic interface. FX News offers necessary functionality but removes unnecessary information, making it a perfect utility for beginner and intermediate foreign exchange traders.

Raiden X

Raiden X is a 2D top down shooter Java game adapted from the popular flash game Raiden X. Java.awt and Java.swing were used in the creation of this game.

Swarm

Swarm is a copy of the game Swarm which is a variation of Space Invaders. Swarm is made using Java.awt and Java.swing.

Tetris Battle

Tetris Battle is a recreation of the popular Facebook Game in Python 2.7 and Pygame. A local two player mode was implemented, allowing two people to play on one keyboard.
